QUICK START
═══════════════════════════════════════════════════════════════════

1. Install dependencies (macOS M1):
   brew install cmake sqlite lua

2. Build the engine:
   cd /Users/romandivkovic/repos/basketball-manager
   ./build.sh release

3. Verify build:
   ls -la build/lib/libBMCore.dylib
   ls -la build/bin/bm-core

DETAILED SETUP
═══════════════════════════════════════════════════════════════════

Step 1: Prepare Build Environment
─────────────────────────────────

mkdir -p /Users/romandivkovic/repos/basketball-manager/build
cd /Users/romandivkovic/repos/basketball-manager/build

Step 2: Configure with CMake
──────────────────────────────

cmake -DCMAKE_BUILD_TYPE=Release \
      -DCMAKE_OSX_ARCHITECTURES=arm64 \
      -DCMAKE_OSX_DEPLOYMENT_TARGET=11.0 \
      ..

Step 3: Build
──────────────

make -j4

Or use the convenience script:
./build.sh release

TROUBLESHOOTING
═══════════════════════════════════════════════════════════════════

Issue: CMake not found
Solution: brew install cmake

Issue: SQLite3 not found
Solution: brew install sqlite

Issue: Lua not found
Solution: brew install lua

Issue: Build fails with architecture errors
Solution: Make sure you're on M1 Mac and use:
  cmake .. -DCMAKE_OSX_ARCHITECTURES=arm64

Issue: libBMCore.dylib not loading in Unity
Solution: Verify file exists and is ARM64:
  file build/lib/libBMCore.dylib

PROJECT STRUCTURE
═══════════════════════════════════════════════════════════════════

bm-core/          C++ Simulation Engine (12 headers + 12 implementations)
bm-ui/            Unity C# Bridge (P/Invoke layer)
bm-data/          Game Data (schema, NCAA rules, player database)
bm-tools/         Utility Tools (data importer, mod validator)
mods/             Mod Examples (example-mod template)
docs/             Documentation (architecture, modding guide)
build/            Generated build artifacts (not in git)

KEY FILES
═══════════════════════════════════════════════════════════════════

CMakeLists.txt              - Root build config
bm-core/CMakeLists.txt      - Engine build config
bm-core/include/BasketballManager.h - Main API header
bm-data/schema/schema.lua   - Database schema
bm-data/ncaa-2025/ncaa_rules.lua - NCAA rules
mods/example-mod/mod.json   - Mod manifest example
